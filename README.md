## Overview

MinorMap Plus is a smart web-based platform designed to help students discover and choose the most suitable **minor specialisation** based on their interests, strengths, career goals, and academic background. Instead of blindly selecting a minor, students receive data-driven recommendations through an intuitive and guided experience.

This project bridges the gap between confusion and clarity in academic decision-making.

## How It Works

1. User signs up / logs in
2. Completes a guided questionnaire
3. System analyses responses
4. Best-fit minor options are generated
5. User receives detailed suggestions with reasoning
